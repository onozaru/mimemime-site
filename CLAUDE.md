# mime mime — SNS投稿ジェネレーター

就労移行支援ブランド **mime mime** のSNS投稿を、Yahoo知恵袋スタイルで生成するプロジェクトです。

## クイックスタート（Claude Code から）

```
SNS投稿を1本作って
```
→ `themes.json` の未使用テーマからランダムに1本生成して `output/posts.md` に保存します。

```
SNS投稿を5本まとめて作って
```
→ 5本バッチ生成。

```
テーマ「休職から復職するのが怖い」でSNS投稿を作って
```
→ テーマを指定して生成。

```
X用にSNS投稿を作って
```
→ プラットフォームを指定（x / note / instagram）。

## スクリプト直接実行（API バッチ）

```bash
pip install anthropic
export ANTHROPIC_API_KEY=sk-ant-...

python generate.py               # 未使用テーマから1本
python generate.py 5             # 5本まとめて
python generate.py --theme "休職から復職するのが怖い"
python generate.py --platform x  # x / note / instagram
```

## ファイル構成

| ファイル | 役割 |
|---|---|
| `generate.py` | メイン生成スクリプト |
| `themes.json` | テーマ一覧（カテゴリ×テーマ名） |
| `progress.json` | 使用済みテーマの記録（自動生成） |
| `output/posts.md` | 生成された投稿の蓄積先（自動生成） |

## テーマカテゴリ

- 就職・転職
- 職場での困りごと
- 休職・復職
- セルフケア・メンタル
- 支援機関・制度
- ADHD・発達障害
- うつ・精神疾患

新しいテーマを追加するには `themes.json` の `themes` 配列に `{"category": "カテゴリ名", "theme": "テーマ名"}` を追加してください。

## 投稿フォーマット

```
─────────────
テーマ：{カテゴリ} / {テーマ名}
日付：YYYY-MM-DD

【質問】
当事者一人称の相談文（120〜200字）

【会話】
A：…
B：…
C：…

【ベストアンサー】★
共感 → 具体的な一歩 → 希望

― ベストアンサーに選ばれました ―

#ハッシュタグ #就労移行支援 #障害者雇用
─────────────
```

## ブランドトーン

- あたたかく誠実、煽らない、上から目線にしない
- 「うまく話せなくていい」の世界観
- 絵文字は0〜2個まで
- 医療・福祉の断定は行わない（「必ず治る」等は禁止）
- ベストアンサーは「共感 → 具体的な小さな一歩 → 希望」の順
